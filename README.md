# Unity_The_Blade_of_Souls_Team_Project
The Blade of Souls: Path of the Red Moon

## Repository 관리 규칙

1. 기본 브랜치는 `main`입니다.
2. 기능 개발 담당자는 `dev_logic` 브랜치에서 작업 후 `main`에 반영합니다.
3. 작업 시작 전 항상 최신 `main`을 `pull`합니다.
4. 개인 작업은 `Assets/_Workspace/본인이름` 폴더에서 진행합니다.
5. 공용 리소스는 `Assets/_Core` 폴더에 위치합니다.
6. 작업 완료 후 결과물만 공용 폴더로 이동합니다.
7. 같은 Scene 또는 Prefab을 여러 사람이 동시에 수정하지 않습니다.
8. 공용 파일 수정 시 팀에 공유 후 작업합니다.
9. 커밋은 작업 단위로 나누어 진행합니다.
10. `main` 브랜치는 항상 정상 실행 가능한 상태를 유지합니다.

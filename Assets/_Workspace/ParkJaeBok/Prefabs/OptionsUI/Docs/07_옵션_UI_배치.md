# 6. 탭별 Row 배치

## Display

- Screen Mode
    - 화면 모드 - Enum
    - 해상도 - Enum
- Sync & Frame
    - VSync - Enum
    - 프레임 제한 - Enum
- Brightness
    - Gamma / Brightness - Numeric
- Graphics Quality
    - 그래픽 품질 프리셋  - Enum
    - 개별 품질 옵션 - Action

## Audio

- Volume
    - Master - Numeric
    - BGM - Numeric
    - SFX - Numeric

## Input

- Input Behavior
    - 입력 유지 방식 변경 - Enum
- Key Binding
    - 입력 설정 - Action

## Accessibility

- Subtitle
    - 자막 on/off - Enum
    - 자막 크기 - Enum
    - 자막 속도 - Numeric
- UI Readability
    - UI 크기 - Numeric
- Motion & Screen Effect
    - 카메라 흔들림 - Numeric
    - 화면 흔들림 - Numeric
    - 흔들림 감소 - Numeric
    - 깜빡임 효과 감소 - Enum
- Visual Accessibility
    - 색약 보정 - Enum
    - 고대비 모드 - Enum

## Gameplay

- Gameplay
    - 난이도 - Enum
- Save
    - 자동 저장 알림 표시 - Enum
